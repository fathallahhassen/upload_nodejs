# Node.js Server Project
This project is a basic Node.js server implementation that demonstrates how to create a simple HTTP server, handle requests, and route them to specific handler functions. The server is designed to serve an initial HTML page, handle file uploads, and display the uploaded image. The project showcases a fundamental understanding of Node.js, HTTP servers, and request handling.

## Features
* Handles HTTP requests and routes them to specific handler functions
* Serves an initial HTML page
* Handles file uploads
* Displays the uploaded image
* Uses a modular design for easy maintenance and scalability

## Tech Stack
* Node.js
* HTTP module
* URL module
* Querystring module
* FS module
* Formidable module
* JavaScript

## Installation
To install the project, follow these steps:
1. Clone the repository using `git clone`
2. Navigate to the project directory using `cd`
3. Install the dependencies using `npm install`
4. Start the server using `node index.js`

## Usage
To use the server, follow these steps:
1. Open a web browser and navigate to `http://localhost:3000`
2. Upload a file using the file upload form
3. The server will display the uploaded image
